Savage Punch

This is a machine equipped with sensors which are monitored by Arduino a programable logic controller. It is a stress release model designed for all frustrated students who are overwhelmed with school work. To release stress student can shout and beat at the machine racking up points and measuring their galvanic skin response
This was a school project that was undertaken in order to understand how program microcontrollers and how to use the sensor readings to create an interactive machine.
