Sick Fits

Sick Fits is a full-stack online clothing store that allows users to search, sell, add to cart, and checkout their favorite items. Built using React, Next.js, and styled components, this application features real credit card checkout. The admin interface is powered by Keystone.js, enabling efficient data management. The project also utilizes Apollo Client for GraphQL mutations and queries. The application showcases six main relational models—Users, Items, Orders, CartItems, OrderItems, and Roles—highlighting the capabilities of relational GraphQL queries.
